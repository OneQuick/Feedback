# 欢迎

欢迎使用 **OneQuick - 热键工具**。

OneQuick 没有复杂的配置，各种难记的按键组合。

OneQuick 专注于简单、符合直觉的操作，就像您早已习惯的 Ctrl-C，Ctrl-V 一样。

### 初次使用？

**将鼠标移动到屏幕左上角↖，滚动鼠标滚轮🖱，会发现音量变化了。**

**选中一段文字，复制两次（Ctrl-C-C），再按下B，直接打开了百度搜索结果页。**

恭喜您，已经入门了OneQuick。
